자동 분류 기준
받아쓰기 오류 유형 분류표
AI가 학생 답안과 정답을 비교해 다음 유형 중 하나로 자동 분류합니다.
오류 유형예시 (정답 → 오답)해당 단계지도 방법
받침 누락학교 → 하교2단계받침 인식 강화, 음절 분리 활동
받침 혼동밖 → 박2단계유사 받침 비교 카드
겹받침 오류닭 → 닥, 흙 → 흑2단계겹받침 단어 모음집 반복
연음 오류꽃이 → 꼬치3단계"소리는 같아도 쓰기는 달라요" 비교 학습
된소리 오류학교 → 학꾜, 갑자기 → 갑짜기3단계발음과 표기 분리 인식
구개음화같이 → 가치3단계형태소 인식 지도
띄어쓰기학교에간다 → 학교에 간다4단계문장 단위 분석
기타모음 혼동, 자음 혼동 등1~2단계발음·시각 변별 활동
